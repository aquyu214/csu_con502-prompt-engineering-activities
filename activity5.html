Scenario (Real CDC FACE Case)

This activity uses a real construction accident case from the CDC FACE Program. Please review the case here:
https://stacks.cdc.gov/view/cdc/164823

Scenario summary:

An ironworker was installing rebar in a five-foot trench next to an existing cinder-block retaining wall. The trench had been excavated below the base of the retaining wall, and there was no shoring, bracing, or structural support for the adjacent wall. The retaining wall collapsed suddenly into the trench, fatally striking the worker.

Your Task

Use any prompt engineering techniques you'd like to generate an image that realistically represents this accident scenario.

You may use:

  • Standard prompting
  • Chain-of-Thought prompting
  • Automatic Prompt Engineering (APE)
  • Prompt rewriting / prompt mutation
  • Detail enhancement / constraint-based prompting

Your goal is to produce the most realistic, accurate, and detailed visual scene.
